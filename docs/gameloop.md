# Gameloop
## Typical Match
Typical match looks like this:
- join game, land in a pod in a small region with the rest of your team
- kill mobs while gathering materials for ammo and buildings
- build a mine
- build a turret
- build power poles / pylons to connect them
- fight off big wave of mobs
- when more quiet, build more and expand the operation
- see that a wave of enemies is going to appear near your first mine
- decide it's likely not defended enough, so rush back to fight the enemies
- quickly build an extra turret to help fight
- once the wave is over, move the turrets into a better position and upgrade some of the other defenses in the region
- reach a certain amount of a special mined material (cannot be hand-mined)
- extraction point appears
- fight big wave of enemies to get to extraction point
- end of match, appear back at lobby
