# Buildings
### Satlink
- while present, allows for making new buildings
- can only be placed once in a world
- enemies will try to destroy it

### Platform
- 3x3?
- has walkable nodes on top, and supports on every edge but hollow in between
- costs a small amount or metal
- not placeable above other platforms or on top of walls, must be connected to the ground

### Mine
- uses octate to harvest raw materials (octate mines are free to run)
- cheap to build
- max per team: 10
- harvests whatever is below it, based on each node below (2x2)
- need two octate worth to run one mine

### Barricade
- forms points in a row
- when built, nodes appear there
- only in straight lines
- enemies can damage these walls, and will often do so instead of walking around
- can place other buildings (not more walls) on top, such as turrets

### Turrets
#### Gatling
- consistent single target damage
- cheap to build
- low ammo use

#### Mortar
- normal ammo use
- AoE damage

#### Railgun
- takes time to aim, only targets armored units
- high ammo use
- expensive
